# pza-plugin-vi
Panduza Virtual Instruments

```bash
cargo install cargo-post
cargo post build --features plugin
```




# full usage


```json
{
    "devices": [
        {
            "name": "Virtual REPL",
            "dref": "vi.repl"
        },
        {
            "name": "Attribute Tester",
            "dref": "vi.tester"
        }
    ]
}
```
